IHttp 基于反射和注解的c++ http 框架

  • IHttp 是一款基于c++的快速开发，高效运行的 网络框架。 它降低c++网络开发的难度，并提高开发效率。
  • 框架提供了和 Spring 一致的操作逻辑和Api。
  • 基于反射和注解，实现了路由映射，参数注入，请求拦截，任意返回值类型处理，异常处理等功能。
  • 对于 query, path, cookie, header, session, form-data, multipart, json, xml 等请求参数的处理，均可以快速处理。
  • 目前插件实现了对 session, 静态文件映射，模板渲染等功能。
  • 文档: https://docs.iwebcore.org/IHttpCore/overview/

IRdb 基于注解的c++数据库框架

  • 操作类似于 Spring Data JPA, 实现了数据库操作的自动化，简化开发难度。
  • 基于注解，实现了数据库表/视图的自动映射，自动创建，json转换
  • 基于model 框架，实现了数据库的查询，更新，删除，分页，排序， 事务处理等功能。
  • 目前支持 sqlite, mysql, sqlserver, Postgres, MariaDb 五种数据库，后续会有更多的数据库被支持。
  • 支持用户自定义扩展，可以实现复杂的查询，更新，删除，分页，排序等功能。
  • 文档：https://docs.iwebcore.org/ICmd/overview/

ICmdCore 基于反射和注解的c++命令行框架

  • 基于反射和注解，实现了命令行参数解析，参数前后拦截，命令执行，命令帮助等功能。
  • 自定义命令格式
  • 文档: https://docs.iwebcore.org/ICmd/overview/
