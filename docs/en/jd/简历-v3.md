## Other

- About the Open-Source Server Framework **IWebCore**

  I have been developing this framework intermittently for over two years, planning to release the first complete version by the end of this year.

  The goal in creating this framework was to establish a C++ server framework comparable to Spring/Flask, with self-inspection capabilities, annotation-driven configuration for generating various functionalities, and decoupling of modules through inversion of control to enhance development efficiency.

  **IWebCore** emulates various features of Spring. It uses macros + QtProperty to mimic annotation functionality in Java/Python, QtGadget for type parsing, RAII for dynamic configuration injection, static type initialization + CRTP for module injection, and Qt reflection for Controller name, parameter parsing, and matching...
