## Other

- About the Open-Source Server Framework `IWebCore`

    This software framework has been under development intermittently over more than two years. The first complete version is planned to be released by the end of this year.

    The goal of developing this framework was to create a server framework for `C++` that could rival `Spring`/`Flask`, featuring self-reflection capabilities, annotation-driven information generation, and decoupling of functional modules via inversion of control, thereby making development convenient and efficient.

    `IWebCore` emulates various functionalities of `Spring`. It mimics Java/Python annotation functionality using `Macros` + `QtProperty`. Type parsing is achieved through `QtGadget`. Dynamic configuration injection uses `RAII`. Module injection leverages `Static Initialization` and `CRTP`. Controller name, parameter parsing, and matching are handled via `Qt` reflection....
