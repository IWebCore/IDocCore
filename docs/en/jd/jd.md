#### IHttp C++ HTTP Framework based on Reflection and Annotations

- IHttp is a C++ framework for rapid development and efficient operation. It reduces the complexity of C++ network development and improves development efficiency.
- The framework provides consistent operational logic and API with Spring.
- Based on reflection and annotations, it implements route mapping, parameter injection, request interception, handling of arbitrary return types, and exception handling.
- It efficiently handles various request parameters such as query, path, cookie, header, session, form-data, multipart, JSON, and XML.
- Currently supports plugins for session management, static file mapping, and template rendering.
- Documentation: [https://docs.iwebcore.org/IHttpCore/overview/](https://docs.iwebcore.org/IHttpCore/overview/)

#### IRdb C++ Database Framework based on Annotations

- Similar to Spring Data JPA, it automates database operations and simplifies development.
- Based on annotations, it automatically maps and creates database tables/views, and handles JSON conversion.
- Based on the model framework, it implements database query, update, delete, pagination, sorting, and transaction management.
- Supports SQLite, MySQL, SQL Server, PostgreSQL, and MariaDB. More databases will be supported in the future.
- Allows user-defined extensions for complex query, update, delete, and pagination/sorting operations.
- Documentation: [https://docs.iwebcore.org/IRdb/overview/](https://docs.iwebcore.org/IRdb/overview/)

#### ICmdCore C++ Command-Line Framework based on Reflection and Annotations

- Based on reflection and annotations, it implements command-line argument parsing, parameter interception, command execution, and command help.
- Customizable command format.
